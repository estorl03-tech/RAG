🛰️ Space-Tech Q&A System (RAG)

📌 概要


このプロジェクトは、特定のWebサイト（宇宙技術関連）から情報を取得し、OpenAIの埋め込みモデルとベクトル検索を組み合わせて、精度の高い回答を生成する RAG (Retrieval-Augmented Generation) システムです。

AIが事前学習していない最新の外部データや、特定のドメイン知識（この場合は宇宙太陽光発電など）に基づいた回答が可能です。


✨ 主な機能

Webスクレイピング: BeautifulSoup を使用し、特定のWordPressサイトから記事本文を抽出。

インテリジェント・クレンジング: キーワードベースで記事の開始・終了位置を特定し、広告やメニューなどのノイズを排除。

ドキュメントの分割（Chunking）: 長文を適切なサイズ（1000文字、100文字重複）に分割し、検索効率を最適化。

ベクトル検索: text-embedding-3-small を使用してテキストをベクトル化し、コサイン類似度で質問に最適なコンテキストを特定。

根拠に基づく回答生成: 抽出した情報をGPT-4o-miniに渡し、ハルシネーション（嘘）を抑えた回答を生成。


🛠 技術スタック

Language: Python 3.12

AI / LLM: OpenAI API (gpt-4o-mini, text-embedding-3-small)

Main Libraries:

openai: AIモデルの呼び出し

beautifulsoup4: Webスクレイピング

scikit-learn: コサイン類似度の計算

numpy: ベクトル演算


🚀 実行方法（Google Colab）

ファイルの準備: GitHub上の .ipynb ファイルを Google Colab にアップロードしてください。

APIキーの設定: Colab左側の鍵アイコン（シークレット）をクリックし、OPENAI_API_KEY を登録して「ノートブックからのアクセス」をONにしてください。

実行: 上から順にセルを実行します。最後のセルで質問（例：「宇宙太陽光発電とは？」）を投げると、Webサイトの情報を元にAIが回答します。


📊 実行結果の例

質問: 「宇宙太陽光発電とはどんな計画ですか？」 回答: > 宇宙太陽光発電は、「宇宙太陽光利用システム（SSPS）」として知られ、静止軌道上で太陽光を集め、それを地上へ送る構想です。このシステムでは、宇宙空間で集めた太陽エネルギーを効率的に利用し、電力を地上に供給することが目指されています。
